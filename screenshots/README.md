# Screenshots

This directory contains screenshots for the gPlantUML project.

## Required Screenshots

1. **class-diagram.png** - Main screenshot showing class diagram (referenced in main README)
   - Should show: Split view, outline panel, syntax highlighting, preview

## Optional Screenshots

2. **sequence-diagram.png** - Sequence diagram with the template example
3. **dark-mode.png** - Dark mode showcase
4. **template-gallery.png** - Template gallery dialog
5. **ai-assistant.png** - AI assistant dialog
6. **all-diagram-types.png** - Grid showing all 10 diagram types

## Screenshot Guidelines

- **Resolution:** At least 1200x800 for main screenshot
- **Format:** PNG with transparency if possible
- **Content:** Clean examples, avoid personal data
- **UI:** Show key features (outline panel, syntax highlighting, real-time preview)

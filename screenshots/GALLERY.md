## Notes

Screenshots demonstrate:
- **Professional color themes** across all diagram types
- **Consistent visual language** between PlantUML and Mermaid
- **Advanced features** like styling, interactivity, and organization
- **Beautiful default appearance** out of the box
- **Dark mode support** with system theme integration

All diagrams rendered with **gDiagram v0.1.0** using native Graphviz rendering.

**Quality**: Production-ready, professional output suitable for:
- Technical documentation
- Presentations
- Academic papers
- Web content
- Print materials

---

*To generate your own screenshots, use gDiagram's export functionality:*
```bash
gdiagram mydiagram.mmd
# Then: Ctrl+Shift+P for PNG, Ctrl+Shift+S for SVG
```
